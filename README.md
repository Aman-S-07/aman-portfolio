## 🚀 Deployment using Ansible

### Step 1: Install Ansible

```bash
sudo apt update
sudo apt install ansible -y
```

### Step 2: Configure Inventory

```ini
[web]
your_server_ip ansible_user=your_user
```

### Step 3: Run Playbook

```bash
ansible-playbook -i inventory playbook.yml
```
